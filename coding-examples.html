Coding Examples

Sass Loop

This loop, written in Sass, defines the theme colours of the nav bar in my recreation of the Netmatters homepage. It does so by accessing theme colours defined in a Sass map in the _config file, allowing someone to easily modify these colours should they desire. Using a loop to do this rather than individually coding each item adheres to the Don't Repeat Yourself, (DRY), principle which is general coding best practice.

Javascript Menu Icon

This Javascript function is taken from this website and causes the menu pop-out icon to appear and dissapear as the user scrolls It accomplishes this by adding a CSS class to the hamburger element when the user has scrolled it out of viewport which brings it back into viewport. The icon will appear for two seconds before dissapearing again, unless the user is hovering on the item.
